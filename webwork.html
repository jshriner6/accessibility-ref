    Section 7 WeBWorK

    Use Case.

    Author exercises that can provide immediate feedback.
    🔗
    🔗

    Time Commitment.

    Moderate to significant. This will depend if you are creating all of your content from scratch, or if you are modifying existing content.
    🔗
    🔗

    Subsection 7.1 WeBWorK Exercises

    WeBWorK
     1 
    webwork.maa.org/moodle/
    is a web-based system that allows you to write and assign self-graded questions. Student progress can be tracked and imported into Canvas. The CSU WeBWorK server
     2 
    wuzzy.natsci.colostate.edu/webwork2
    has a list of existing courses in WeBWorK that can be used and modified.
    🔗
    There are built-in accessibility features in WeBWorK
     3 
    webwork.maa.org/moodle/mod/page/view.php?id=29
    . The main things to be thoughtful of when writing or using a WeBWorK exercise are the following.
    1. All math content should be authored in math mode (i.e., enclosing math in \( \)).
      🔗
      🔗
    2. If your exercise contains an image, be sure to write alternate text that includes the information needed from the image to answer the question being asked. If you are writing or using questions that have variation, be sure to make your alternate text dynamic by using the necessary variables in your image description.

      🔗
